// Example: Find unique string in array
function findUniq(arr) {
  const normalize = str => [...new Set(str.toLowerCase().replace(/\s+/g, '').split(''))].sort().join('');
  const signatures = arr.map(normalize);
  for(let i=0;i<signatures.length;i++) {
    if(signatures.indexOf(signatures[i]) === signatures.lastIndexOf(signatures[i])) return arr[i];
  }
}

Pixel Panic: obstacle avoidance game

Pixel Panic is a charmingly simple yet surprisingly addictive obstacle avoidance game where you steer a tiny pixel block left and right to dodge a relentless barrage of falling obstacles. Starting with 5 lives, the challenge ramps up across three levels as the swarm grows denser and speeds pick up, testing your reflexes and nerves. With its retro pixel vibe and no-nonsense mechanics, it’s the perfect quick fix for a burst of casual gaming fun—easy to pick up, tough to put down, and guaranteed to send your heart racing in the best pixelated panic.

Aims of the game

  • Player movement (left/right with Arrow keys).
  • Obstacles (pixels) fall from the top at increasing frequency & speed.
  • A lives system (5 hearts) disappears when hit; collision causes a live lost.
  • 3 Levels: increasing levels become harder with more pixels spawned and the speed of appearance increasing!
  • Game Over when lives reach 0.
  • Collectible stars to score points.
  • Score counted (points based on survival &/or collected stars).
